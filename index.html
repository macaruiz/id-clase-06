Índice de Materialidad Irrecuperable Según Región

Macarena Ruiz, Irene Castiglione y Celeste Somervell

Según el CENSO de 2017, el Índice de Materialidad expone las condiciones materiales de las viviendas de Chile, a partir de los materiales predominantes en paredes exteriores, cubierta de techo y pisos.

Este índice se puede declarar como recuperable o irrecuperable. Las siguientes condiciones determinan si este índice es irrecuperable:

  • Si las paredes exteriores cuentan con materiales precarios como lata, cartón, plástico, entre otros.
  • Si la cubierta del techo presenta materiales precarios como lata, cartón, plástico, entre otros.
  • Si no posee cubierta sólida en el techo.
  • Si el piso se declara como tierra.
